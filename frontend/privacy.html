Privacy Policy

Template — review with legal counsel before launch.

What we store

Email, password (hashed by PocketBase), Telegram handle, subscription status, imported bot definitions, and audit logs. Broker tokens, when connected, are stored server-side and never exposed to the browser.

Email

We use Resend to send verification, password reset, and account emails.

Security

Sensitive data and all role/permission checks live on the backend. API keys and admin credentials are never present in frontend code.
